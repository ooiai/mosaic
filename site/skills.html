Skills Module: Step-by-Step Tutorial

Use skills as reusable execution knowledge units. This page focuses on lifecycle operations.

Step 1: Discover installed skills

mosaic --project-state skills list
mosaic --project-state skills list --source all

Step 2: Inspect skill metadata

mosaic --project-state skills info <skill_id>
mosaic --project-state skills check <skill_id>

Step 3: Install skill from local path

mosaic --project-state skills install --path ./skills/example-skill
# overwrite existing installation
mosaic --project-state skills install --path ./skills/example-skill --force

Step 4: Validate all skills

mosaic --project-state skills check

Step 5: Remove stale skill

mosaic --project-state skills remove <skill_id>
mosaic --project-state skills list

Step 6: JSON output for scripts

mosaic --project-state --json skills list
mosaic --project-state --json skills check
