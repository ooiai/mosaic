Overview

Mosaic CLI documentation

Mosaic is a Rust-first local agent runtime and operations CLI. This docs site is focused on practical setup, profile configuration, and day-to-day diagnosis.

Scope: CLI-first delivery. Web/Desktop shell is not part of this phase.

What is available now

  • Core: setup, configure, models, ask, chat, session
  • Ops: status, health, doctor, logs, system
  • Runtime modules: channels, gateway, approvals, sandbox, memory, security

First commands

mosaic --project-state setup \
  --base-url https://api.openai.com \
  --api-key-env OPENAI_API_KEY \
  --model gpt-4o-mini

mosaic --project-state models list
mosaic --project-state ask "summarize this repository"
mosaic --project-state chat
