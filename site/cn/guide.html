路径 A：10 分钟上手

适用于首次使用，或在新机器上快速验证环境。

  • 安装并验证 mosaic --help。
  • 用一个 profile 跑通 setup。
  • 通过 models list 和 health 验证连通性。
  • 执行 ask 和 chat。

路径 B：生产运维

适用于团队/长期运行场景，强调可观测、可诊断、可恢复。

  • 配置并验证消息通道。
  • 运行网关并持续探测健康状态。
  • 启用 approvals + sandbox 策略。
  • 使用 status/health/doctor/logs/dashboard 做日常巡检。

路径 C：深度学习与模块专项

适用于需要系统化掌握能力细节的场景。

  • 按 分阶段学习 从入门走到高级。
  • 按 Models 与 Profiles 固化模型路由与配置基线。
  • 按 Sessions 教程 建立可回放会话流程。
  • 按 TUI 教程 使用 chat-first 终端界面并掌握快捷键流程。
  • 按 策略教程 建立 approvals/sandbox 安全基线。
  • 按 Azure 运维剧本 一次性跑通生产基线。
  • 按 Agents 专项 完整掌握 agent 与路由。
  • 按模块教程补齐 Channels、Memory、Knowledge、Plugins、Skills、Gateway、Gateway Call API。

建议顺序

  1. 先完成 10 分钟上手。
  2. 再走完 分阶段学习。
  3. 补完 Agents 专项。
  4. 进入 生产运维。
  5. 发版前执行 回归测试教程中的门禁流程。
  6. 把 排障页 作为运行手册兜底。
