总览

Mosaic CLI 文档站

Mosaic 是 Rust 优先的本地 Agent 运行时与运维 CLI。本文档聚焦安装、配置、日常运行和故障定位。

范围说明：当前阶段为 CLI 核心能力，不包含 Web/Desktop 外壳交互。

当前命令范围

  • 核心命令：setup、configure、models、ask、chat、session
  • 运维命令：status、health、doctor、logs、system
  • 运行模块：channels、gateway、approvals、sandbox、memory、security

首批命令

mosaic --project-state setup \
  --base-url https://api.openai.com \
  --api-key-env OPENAI_API_KEY \
  --model gpt-4o-mini

mosaic --project-state models list
mosaic --project-state ask "hello"
mosaic --project-state chat
