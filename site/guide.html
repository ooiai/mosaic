Usage tutorial

From install to first agent session

1. Install

# macOS
brew tap ooiai/mosaic https://github.com/ooiai/mosaic
brew install mosaic

# Linux/macOS
curl -fsSL https://raw.githubusercontent.com/ooiai/mosaic/main/cli/install.sh |
bash -s -- --from-source

2. Configure

export OPENAI_API_KEY=your_key
mosaic --project-state setup \
  --base-url https://api.openai.com \
  --api-key-env OPENAI_API_KEY \
  --model gpt-4o-mini

3. Run

mosaic --project-state models list
mosaic --project-state ask "summarize this repository"
mosaic --project-state chat

4. Operate

mosaic --project-state status
mosaic --project-state health
mosaic --project-state doctor
mosaic --project-state channels list
