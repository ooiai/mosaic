Guide

From install to daily usage

Recommended path for new users. Follow this order once, then keep the troubleshooting page open in your workflow.

Step 1. Install binary

Choose your platform method from the install page and verify with mosaic --help.

Step 2. Setup project state

mosaic --project-state setup \
  --base-url https://api.openai.com \
  --api-key-env OPENAI_API_KEY \
  --model gpt-4o-mini

Step 3. Validate provider and model

mosaic --project-state models list
mosaic --project-state health

Step 4. Start using agent runtime

mosaic --project-state ask "hello"
mosaic --project-state chat

Step 5. Operate and diagnose

mosaic --project-state status
mosaic --project-state doctor
mosaic --project-state logs --tail 80
