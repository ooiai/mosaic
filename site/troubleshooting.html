Troubleshooting

Common issues and direct fixes

Run these checks first, then use the table to identify the most likely fix path.

Baseline checks

mosaic --project-state status
mosaic --project-state health
mosaic --project-state doctor
mosaic --project-state logs --tail 80

Issue matrix

Symptom Check Typical fix
404 Not Found on models mosaic configure --show Fix base URL, avoid duplicated /v1 in Azure path
command not found: mosaic which mosaic Add $HOME/.cargo/bin to PATH and restart shell
Provider auth/network errors mosaic health, env vars Check key env name/value and endpoint reachability

Channels and gateway diagnostics

mosaic --project-state channels list
mosaic --project-state channels logs --channel all --tail 50
mosaic --project-state gateway probe
